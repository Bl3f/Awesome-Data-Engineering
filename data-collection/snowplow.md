# Snowplow

> Web event로 시작해서 모든 event data를 수집 가능한 platform이 되려고 함  
> Custom event를 정의해서 활용할 수 있음  
> 데이터를 schema에 맞게 정형화된 형태로 수집 가능한 것이 장점
